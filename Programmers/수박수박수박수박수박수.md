### 수박수박수박수박수박수?

- 문제 : [12922](https://programmers.co.kr/learn/courses/30/lessons/12922)

~~~java
class Solution {
  public String solution(int n) {
      String answer = "";
      for(int i=1; i<=n; i++)
          if(i%2==0){
              answer += "박";
          } else {
              answer += "수";
          };
      return answer;
  }
}
~~~
