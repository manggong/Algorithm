### 정수 제곱근 판별

- 문제 : [12934](https://programmers.co.kr/learn/courses/30/lessons/12934)

~~~javascript
function solution(n) {
    var answer = 0;
    const sqrt = Math.sqrt(n);
    if (Number.isInteger(sqrt)){
        answer = Math.pow(sqrt + 1, 2)
    } else {
        answer = answer -1;
    }
    return answer;
}
~~~
