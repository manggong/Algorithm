### 콜라츠 추측

- 문제 : [12943](https://programmers.co.kr/learn/courses/30/lessons/12943)

~~~javascript
function solution(num) {
    var answer;
    for(let i=0; i<500; i++){
        if(num!=1){
            num % 2 == 0 ? num = num / 2 : num = num * 3 + 1;
        } else {
        return answer = i;
      }
    }
    return answer = -1;
}
~~~
