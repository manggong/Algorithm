### 같은 숫자는 싫어

- 문제 : [12906](https://programmers.co.kr/learn/courses/30/lessons/12906)

~~~javascript
function solution(arr)
{
    var answer = [];
    
    for(let i=0; i<=arr.length; i++){
        var firstindex = arr[i]
        var secondindex = arr[i+1]
        if(firstindex !== secondindex){
            let now = arr[i]
            answer.push(now)
        }       
    }   
    return answer; 
}
~~~
